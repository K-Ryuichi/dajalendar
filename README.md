Dajalendar
===

## overview
お馴染みのダジャレカレンダー
ランダムでコマンドとダジャレが出てくる
カレンダーはおまけ

## install

```
git clone https://github.com/K-Ryuichi/dajalendar.git
cd dajalendar
python setup.py install
```

## usage

```
dajalendar
```
